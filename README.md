# Airbnb Clone - ME*N Stack

This is a project to build a vacation rental platform similar to Airbnb, using the ME*N stack (MongoDB, Express.js, Node.js). The platform allows users to search, book, and list vacation properties.

## Technologies Used
- **MongoDB**: NoSQL database to store user profiles, property listings, and booking details.
- **Express.js**: Web framework to handle HTTP requests, routing, and business logic.
- **Node.js**: JavaScript runtime to handle server-side operations and API management.

## Features
- User authentication (sign-up, login).
- Search and filter properties by location, price, and availability.
- Listing properties with images, descriptions, and pricing.
- Booking system with reservation management.
- Review and rating system for properties.
